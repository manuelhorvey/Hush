# Figma Foundation

The first Hush design file should establish structure before broad screen design begins.

## Pages

```text
00 - Brand
01 - Design System
02 - Components
03 - User Flows
04 - Screens
05 - Prototype
```

## First Screens

Only these screens are required for the first design pass:

1. Welcome
2. Create Identity
3. Home
4. New Conversation
5. Chat
6. Complete Conversation
7. Conversation Completed

## Design Principle

Design for presence, trust, and completion. The interface should make privacy understandable without turning the product into a technical control panel.
